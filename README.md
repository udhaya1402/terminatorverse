# TerminatorVerse

The Cinephile Vault is a modern movie and web series rating and review website.

## Features
- Browse movies and web series
- Search titles
- Rate and review content
- Responsive design

## Technologies Used
- React
- Vite
- HTML
- CSS
- JavaScript

## Installation

bash
npm install
npm run dev
